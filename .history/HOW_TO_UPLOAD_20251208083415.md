### To Extend

To send files to a server, modify `handleFileUpload()` in `src/App.jsx`:

```javascript
const handleFileUpload = async (field, files) => {
  const formData = new FormData();
  files.forEach((file) => {
    formData.append("file", file);
  });

  const response = await fetch("/api/upload", {
    method: "POST",
    body: formData,
  });
};
```
